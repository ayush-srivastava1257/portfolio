Machine Learning-Based Real-Time Congestion Detection System

Developed a real-time congestion detection system using deep learning and computer vision. The system captures images via a camera and estimates congestion levels by counting people in different density scenarios. It utilizes a CNN-based hybrid model, employing YOLOv8 for low-density crowd detection and CSRNet for high-density scenarios. A real-time image capture and processing pipeline was implemented using OpenCV, along with a dynamic switching mechanism to choose the appropriate model based on detected density. The model was optimized for limited computing resources, ensuring fast and accurate predictions, and was tested on Google Colab using datasets like ShanghaiTech and COCO.

Implemented an AI-powered indoor positioning system that leverages real-time CCTV feedback to determine congestion levels and suggest the shortest or least time-consuming path within VIT Chennai. The system integrates a hybrid CNN model (YOLOv8 + CSRNet) to accurately count people in different areas, and a variable heuristic function was designed to consider both distance and congestion while computing optimal paths. The routing system was built using the A* algorithm, dynamically adapting to real-time congestion data to enhance navigation efficiency. Extensive testing was conducted to ensure scalability and reliability within large indoor environments.

Adaptive Flow Control in Network Communication

In this project, an adaptive flow control mechanism is implemented to enhance the efficiency of network communication. The system dynamically adjusts the packet-sending speed of the sender based on the feedback received from the receiver, such as network congestion levels and buffer status. By interpreting the feedback speed as an indicator of network congestion, the sender optimizes its sending rate to avoid network overloads, ensuring smoother communication and improved data throughput. The primary aim of this project is to simulate real-time network scenarios with multiple senders and a single receiver, allowing for the analysis of the adaptive sender's performance in maintaining stable data transfer rates under varying network conditions. The output of this simulation provides insights into how effective the adaptive flow control mechanism is in mitigating congestion and optimizing data flow.
